# Airport Management Information System

## Project Description
This project is designed to manage airport operations using a relational database management system (RDBMS). The system stores and manages information about airplanes, hangars, technicians, airport employees, traffic controllers, aircraft tests, maintenance records, and employee schedules.

The project was developed using MySQL and MySQL Workbench.

---

## Features

- Airplane management
- Plane model tracking
- Hangar management
- Aircraft test records
- Technician expertise tracking
- Maintenance management
- Employee and traffic controller management
- Shift scheduling
- Statistical SQL queries
